# Fonts

Drop these `.ttf` files in this folder to enable the brand typography:

- `BebasNeue-Regular.ttf` — display / headlines ([Google Fonts: Bebas Neue](https://fonts.google.com/specimen/Bebas+Neue))
- `Inter-Regular.ttf`, `Inter-Medium.ttf`, `Inter-SemiBold.ttf` — body / controls ([Google Fonts: Inter](https://fonts.google.com/specimen/Inter))

Names must match the `UIAppFonts` array in `project.yml`. Until present, the app falls back to system fonts automatically (see `DesignSystem/Typography.swift`).
